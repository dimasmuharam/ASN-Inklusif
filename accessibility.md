# Pernyataan Aksesibilitas

**Forum ASN Inklusif** berkomitmen untuk memastikan situs web ini dapat diakses oleh semua orang, termasuk penyandang disabilitas. Kami terus berupaya meningkatkan pengalaman pengguna dengan menerapkan standar aksesibilitas web global.

## Standar Kepatuhan
Situs web ini dirancang agar sesuai dengan **Web Content Accessibility Guidelines (WCAG) 2.1** pada level **AA**.

## Fitur Aksesibilitas
Kami telah menyematkan fitur-fitur berikut untuk memudahkan akses:

1.  **Navigasi Keyboard:** Seluruh menu dan tautan dapat diakses menggunakan tombol `Tab` tanpa mouse.
2.  **Skip-Link:** Tautan tersembunyi "Lewati ke Konten Utama" untuk pengguna pembaca layar (*screen reader*).
3.  **Teks Alternatif (Alt Text):** Semua gambar informatif dilengkapi deskripsi teks.
4.  **Kontras Warna:** Menggunakan kombinasi warna dengan rasio kontras tinggi agar mudah dibaca oleh pengguna *low vision*.
5.  **Dukungan Pembaca Layar:** Struktur HTML semantik (Heading, Landmarks) yang ramah terhadap JAWS, NVDA, dan VoiceOver.
